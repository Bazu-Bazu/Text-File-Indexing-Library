# Text Indexer

## 📌 Описание

Библиотека для индексации текстовых файлов по словам.

Позволяет:
- добавлять файлы и директории в индекс
- искать файлы по слову
- автоматически отслеживать изменения файлов
- работать в многопоточном режиме

---

## 🧱 Технологии

- Java 21
- Maven
- JUnit

---

## 🛠 Сборка проекта

```
mvn clean install
```

После сборки:
- скомпилируется библиотека
- выполнятся тесты
- создастся .jar файл

---

## ▶️ Запуск CLI приложения

```
mvn exec:java -Dexec.mainClass="com.example.cli.Main" -pl indexer-cli
```

---

## 💻 Использование CLI

### Добавить файл
addFile /path/to/file.txt

### Добавить директорию
addDir /path/to/directory

### Поиск
search word

### Выход
exit
